第一个博客
前言这是我的第一个学习博客，记录一下我安装并部署我的个人博客的过程 安装准备安装hexo的本地设备需要安装nodejs、npm和git，其中nodejs是hexo安装需要的环境，建议不要安装最新版本，安装上个版本的稳定版就好，npm会随nodejs安装的时候一起安装，建议更换一下国内安装源，不然会很慢，git是上传和拉取开源仓库文件的工具，部署的时候会使用。这些环境和工具安装网上都有教程，一搜就能找到，我就不多说了。 安装过程依次输入下面的命令就行 123456$ npm install hexo-cli -g## 下面的命令是初始化hexo,可以自己选择一个你要保存博客的位置，再进入CMD输入命令$ hexo init blog$ cd blog$ npm install$ hexo server
